◆ abonent_add()

int abonent_add ( struct Abonent  directory[DIR_SIZE])

Создаёт новую запись об абоненте.

Новая запись помещается в первую свободную (заполненную нулями) ячейку.

Аргументы
[in,out]directoryСсылка на справочник (массив) с абонентами. Размер справочника задаётся в макросе DIR_SIZE.
Возвращает
Код ошибки, где:
  • 0 - всё в порядке.
  • 1 - произошла ошибка во время ввода имени.
  • 2 - произошла ошибка во время ввода второго имени.
  • 3 - произошла ошибка во время ввода телефона.
Необходимо сделать:
Передавать в сообщение об ошибке код ошибки из функции field_input или человекопонятное объяснение, о том что именно вызвало ошибку и что делать, чтобы ошибки больше не возникало.

Удаление абонента.

Удаляет запись по указателю на конкретного абонента. Удаление заключается в заполнении всех полей нулями.

Аргументы
[in]*abonentуказатель на абонента.

◆ abonent_del_interactive()

int abonent_del_interactive ( struct Abonent  directory[DIR_SIZE])

Интерактивное удаление абонента.

Удаляет запись по id абонента. Обёртка для abonent_del.

Аргументы
[in]idНомер записи в справочнике, которую необходимо удалить. Не параметр функции: Интерактивный ввод с клавиатуры.
[in,out]directoryСсылка на справочник (массив) с абонентами. Размер справочника задаётся в макросе DIR_SIZE.
Возвращает
Код ошибки, где:
  • 0 - всё в порядке.
  • 1 - введено не число.
  • 2 - введённое id за пределами возможных значений.
  • 3 - пустая запись. Уже удалено?

◆ abonent_print()

void abonent_print ( struct Abonent  abonent)

Вывод информации об абоненте.

Выводит информацию об абоненте в строку. Поля начинаются со следующих колонок:

  • имя - с 5й колонки;
  • второе имя - с 16й колонки;
  • телефон - с 27й колонки.
Заметки
Предназначена для использования в составе других функций.
Аргументы
[in]abonentабонент, информацию о котором надо вывести.
