◆ abonent_add()

int abonent_add ( struct Directory *  directory)

Создаёт новую запись об абоненте.

Аргументы
[in,out]directoryСсылка на справочник (массив) с абонентами.
Возвращает
Код ошибки, где:
  • 0 - всё в порядке.
  • 1 - произошла ошибка во время ввода имени.
  • 2 - произошла ошибка во время ввода второго имени.
  • 3 - произошла ошибка во время ввода телефона.
Необходимо сделать:
Передавать в сообщение об ошибке код ошибки из функции field_input или человекопонятное объяснение, о том что именно вызвало ошибку и что делать, чтобы ошибки больше не возникало.

◆ abonent_del()

int abonent_del ( struct Directory *  dir,
struct Abonent *  the_abonent 
)

Удаление абонента.

Удаляет запись по указателю на конкретного абонента.

Аргументы
[in]*dirуказатель на справочник.
[in]*the_abonentуказатель на абонента.
Возвращает
  • 0 - всё в порядке
  • -1 - Пустой указатель на справочник, или справочник пуст или пустые указатели на голову/хвост справочника.
  • -2 - Пустой указатель на абонента.

◆ abonent_del_interactive()

int abonent_del_interactive ( struct Directory *  dir)

Интерактивное удаление абонента.

Удаляет запись по имени абонента. Обёртка для abonent_del.

Заметки
Имя абонента для удаления вводится через stdin (консоль).
Аргументы
[in]dirСсылка на справочник (массив) с абонентами.
Возвращает
Код ошибки, где:
  • -1 - Пустой указатель на справочник
  • -2 - Не удалось выделить память под абонента
  • 0 - всё в порядке.
  • 1 - введено не число.
  • 2 - введённое id за пределами возможных значений.
  • 3 - пустая запись. Уже удалено?

◆ abonent_print()

void abonent_print ( struct Abonent  abonent)

Вывод информации об абоненте.

Выводит информацию об абоненте в строку. Поля начинаются со следующих колонок:

  • имя - с 5й колонки;
  • второе имя - с 16й колонки;
  • телефон - с 27й колонки.
Заметки
Предназначена для использования в составе других функций.
Аргументы
[in]abonentабонент, информацию о котором надо вывести.
