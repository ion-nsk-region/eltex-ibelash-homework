Подробное описание

Структура, хранящая имена и телефоны абонентов.

По каждому абоненту в справочнике хранится следующая информация:

  • имя
  • второе имя (фамилия)
  • телефон.

Размер данных не более FIELD_SIZE символов для латиницы и в два раза меньше для кириллицы.
